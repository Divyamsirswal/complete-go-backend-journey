## 🤔 Why This Repo Exists

Most Go tutorials rush to frameworks and skip *why* things work.

This repo is different. Every concept is earned, not skipped. Every line of code is written by hand. Every day I stream my progress live — no edits, no polished tutorials, just real learning in real time.

**If you're a beginner learning Go for backend development**, this repo is your companion:
- Follow the exact same roadmap, day by day
- Read human-written notes for every single day
- Clone and run each day's challenges yourself
- Watch the thinking process, not just the final output

---

## ✅ What You'll Learn by Following This

By the end of this 30-day journey, you'll be able to:

- **Write idiomatic Go** — no bad habits carried from other languages
- **Build REST APIs from scratch** — routing, JSON, CRUD, no heavy frameworks
- **Integrate a real database** — PostgreSQL with raw SQL
- **Implement auth** — login, signup, JWT, protected routes
- **Structure backend projects cleanly** — handlers, middleware, separation of concerns
- **Ship a complete backend API** — auth + DB + CRUD as a final project

**Option 1 — Follow the journey:**


**Option 3 — Read the notes only:**
