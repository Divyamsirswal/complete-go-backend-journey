## Difference between GET and POST ?
* ``GET`` -> use to extract some info from the server, not secure
* ``POST`` -> when we have to send data to the server , secure

## Query vs path parameter ?
* ``query`` -> to filter out , sort, pagination or used for other things 
* ``path`` -> it is a part of URL , you will see these when you have routes
